# 01 — Design system

## Voice
- Designer-first. Materials are facts. Weights, dimensions, country of origin always specified.
- Banned: "premium", "luxury", "exclusive", "curated", "elevate", "experience" (verb), "indulge", "thoughtful".
- Long answers when warranted; short ones when the question is short.

## Type
- Display: EB Garamond (fallback Söhne Breit / Times New Roman).
- Sans: Inter (fallback Söhne).
- Mono: ui-monospace — for product specs.

## Logo
- MH monogram in a 1px-border square, EB Garamond 30px italic. Used at 32×32 in the header.
- Wordmark "Marsalforn Home" in EB Garamond italic alongside.
- Mark concept = `arch` for favicon/OG/apple-touch — references Maltese architecture.
